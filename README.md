# go-hid

Working prototype to prove that you don't need `cgo` to work with HID devices.
Currently, Windows, macOS and Linux are supported.

Developed as part of [go-ctaphid](https://github.com/go-ctap/ctaphid).

## Status

- [x] Windows
  - [x] Enumerate
  - [x] Open
    - [x] Read
    - [x] Write
- [x] macOS
  - [x] Enumerate
  - [x] Open
      - [x] Read
      - [x] Write
- [x] Linux
  - [x] Enumerate
  - [x] Open
     - [x] Read
     - [x] Write
