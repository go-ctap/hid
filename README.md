# go-hid

Working prototype to prove that you don't need `cgo` to work with HID devices.
Currently, only Windows is supported.

Developed as part of [go-ctaphid](https://github.com/savely-krasovsky/go-ctaphid).

## Status

- [x] Windows
  - [x] Enumerate
  - [x] Open
    - [x] Read
    - [x] Write
- [ ] macOS
  - [ ] Enumerate
  - [ ] Open
      - [ ] Read
      - [ ] Write
- [ ] Linux
  - [ ] Enumerate
  - [ ] Open
     - [ ] Read
     - [ ] Write
